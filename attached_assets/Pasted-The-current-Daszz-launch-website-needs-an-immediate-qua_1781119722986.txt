# Replacement 3D Direction: Digital Skin Twin Reveal

Create an elegant **human face-inspired digital skin twin sculpture**.

The result should look like a luxury skincare campaign visual, not a coding demo.

## Main Visual

Build a centered, front-facing sculptural face mask using one of these approaches, in this priority order:

1. If the repository already contains a high-quality optimized face or mask GLB model, reuse it.
2. If no suitable asset exists, create a procedural face-inspired mask using Three.js geometry with smooth contours.
3. If a genuinely polished procedural 3D mask cannot be achieved, use an elegant layered 2.5D SVG face-line composition with subtle depth, parallax, and WebGL lighting rather than showing a low-quality 3D object.

Quality is more important than forcing a bad 3D model.

## Desired Appearance

The visual should include:

* A symmetrical face-mask silhouette
* Smooth cheek, forehead, jawline, and nose contours
* Thin wireframe lines or fine point-cloud detailing
* Very subtle blue-indigo rim lighting
* Soft inner glow
* A slow rotation of no more than a few degrees
* Gentle breathing-like movement
* Small scan-line movement from top to bottom
* A restrained pointer-follow parallax effect
* A soft spotlight behind the face
* Subtle depth without visual clutter

The face should appear partially revealed from darkness.

## Reveal Progress Effect

Connect the visual subtly to the countdown progress.

As the 36-hour countdown progresses:

* Increase the visible opacity of the face sculpture slightly.
* Reveal additional contour lines.
* Move a scanning highlight gradually across the face.
* Increase the ambient glow gently.
* Never make the scene visually noisy.

At 100% completion, show a refined final reveal state with the full sculpture visible and the text:

> DASZZ IS NOW LIVE
> Your skin. Understood.

## Hero Composition

Improve the hero layout so it feels intentional:

### Desktop

* Left side: reveal label, headline, supporting copy, countdown, CTA
* Right side: digital-skin-twin face sculpture
* Use balanced spacing and avoid unused blank areas

### Mobile

* Reduce complexity
* Keep text readable

* Lazy-load the visual
* Use a static fallback while it loads

* Avoid large external assets
* Avoid page-load layout shift
* Keep animation smooth on mid-range Android devices
